# 国际化设计文档

> 版本：v0.1  
> 范围：PC 管理后台国际化基础能力  
> 当前支持语言：中文、英文

## 1. 设计目标

国际化能力需要支持：

- 中文与英文切换。
- 页面文案集中管理。
- 用户选择语言后本地持久化。
- 后续扩展更多语言。
- 后续支持后端错误消息、枚举字典、菜单名称国际化。

## 2. 技术选型

前端使用：

```text
vue-i18n
```

语言包位置：

```text
apps/admin-web/src/locales
├── index.ts
├── zh-CN.ts
└── en-US.ts
```

## 3. 语言编码

当前约定：

| 语言 | locale |
| --- | --- |
| 简体中文 | zh-CN |
| 英文 | en-US |

## 4. 使用方式

组件中使用：

```ts
const { t } = useI18n()
```

模板中使用：

```vue
{{ t('nav.dashboard') }}
```

## 5. 状态管理

语言状态由 `apps/admin-web/src/stores/app.store.ts` 管理。

本地缓存 key：

```text
shop-saas-locale
```

## 6. 当前接入范围

已接入：

- 应用布局
- 侧边栏菜单
- 登录页主要文案
- 工作台主要文案
- 租户页主要文案

后续需要逐步接入：

- 表格状态文案
- 表单校验文案
- 后端错误消息
- 权限菜单动态文案
- 枚举字典显示文案

## 7. 后续扩展建议

后续可以扩展：

- 根据浏览器语言自动选择默认语言。
- 后端返回错误码，前端按错误码翻译。
- 租户级语言偏好。
- 用户级语言偏好。
- 日期、金额、数字格式本地化。
